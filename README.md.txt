Expense Report Project

Project Description
 Sign in to create expense reports to receive refunds for work expenses. 
 Submit an image of your recipt for review.
 Managers can also sign in to approve reports.

Technologies Used
 Servlets, Java, JavaScript, HTML, CSS, JDBC, SQL, AJAX, Bootstrap, RDS, Tomcat, Git, Maven

Features
 Submit expense reports through image upload. 
 As admin approve or deny requests for reimbursements.

Getting Started
 to host locally, open the package in your IDE, and select run on server. Then navigate to localhost:4200 in your browser to log in

Usage
 - log in with email and password
 - submit expense reports by submitting an amount and image of the recipt
 - admin's have access to a panel that will allow them to approve or deny any expense reports displayed
